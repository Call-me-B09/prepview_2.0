# OCR Service Environment Variables Guide

This file serves as a guide for the environment variables required by the OCR Service.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `OCR_api_key` | The API Key used for accessing OCR.space endpoints. | `your_ocr_space_api_key_here` | Yes |
