# DB Service Environment Variables Guide

This file serves as a guide for the environment variables required by the Database (DB) Service.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `PORT` | The port number on which the Database microservice runs locally. | `5000` | Yes |
| `Mongodb` | MongoDB connection URI to locate the MongoDB instance. | `mongodb://127.0.0.1:27017` | Yes |
