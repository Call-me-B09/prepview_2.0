# Assembly Service Environment Variables Guide

This file serves as a guide for the environment variables required by the Assembly Service.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `Assembly_api` | API Key/token for AssemblyAI used to transcribe audio streams/files. | `your_assembly_ai_api_token` | Yes |
| `PORT` | The port number on which the Assembly microservice runs locally. | `5002` | Yes |
