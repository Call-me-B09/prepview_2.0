# Orchestration Service Environment Variables Guide

This file serves as a guide for the environment variables required by the Orchestration Service. These variables route requests from the Gateway to appropriate downstream microservices.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `PORT` | The port number on which the Orchestration Service gateway runs locally. | `4000` | Yes |
| `OCR_SERVICE_URL` | Endpoint URL of the OCR microservice. | `http://localhost:3000` | Yes |
| `DB_SERVICE_URL` | Endpoint URL of the DB microservice. | `http://localhost:5000` | Yes |
| `AI_SERVICE_URL` | Endpoint URL of the AI microservice. | `http://localhost:5001` | Yes |
| `ASSEMBLY_SERVICE_URL` | Endpoint URL of the Assembly microservice. | `http://localhost:5002` | Yes |
| `TTS_SERVICE_URL` | Endpoint URL of the TTS microservice. | `http://localhost:5003` | Yes |
