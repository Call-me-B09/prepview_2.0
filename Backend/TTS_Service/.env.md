# TTS Service Environment Variables Guide

This file serves as a guide for the environment variables required by the TTS (Text-to-Speech) Service.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `PORT` | The port number on which the Text-to-Speech (TTS) microservice runs locally. | `5003` | Yes |
