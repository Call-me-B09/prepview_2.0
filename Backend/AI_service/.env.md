# AI Service Environment Variables Guide

This file serves as a guide for the environment variables required by the AI Service.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `groq_api` | API key used for accessing Groq Cloud services for running LLMs (e.g. Llama). | `gsk_your_groq_api_key_here` | Yes |
| `PORT` | The port number on which the AI microservice runs locally. | `5001` | Yes |
