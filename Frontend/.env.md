# Frontend Environment Variables Guide

This file serves as a guide for the environment variables required by the Frontend application. Since this repository is pushed to GitHub, do not place actual sensitive credentials here.

## Environment Variables

| Variable Name | Description / Purpose | Example / Format | Required |
| :--- | :--- | :--- | :--- |
| `VITE_DB_SERVICE_URL` | Endpoint URL of the Database Service used to persist user credentials, sessions, and question records. | `http://localhost:5000/db` | Yes |
| `VITE_ORCHESTRATION_SERVICE_URL` | Endpoint URL of the Orchestration Service which coordinates API requests to backend microservices. | `http://localhost:4000` | Yes |
| `VITE_FIREBASE_API_KEY` | Firebase web client API key for authorization and authentication services. | `AIzaSyB3...` | Yes |
| `VITE_FIREBASE_PROJECT_ID` | Google Cloud/Firebase project identifier. | `your-firebase-project-id` | Yes |
| `VITE_FIREBASE_STORAGE_BUCKET` | Storage bucket URI for uploading and hosting documents/files. | `your-project-id.firebasestorage.app` | Yes |
| `VITE_FIREBASE_AUTH_DOMAIN` | Auth Domain identifier used for user auth redirecting. | `your-project-id.firebaseapp.com` | Yes |
| `VITE_FIREBASE_MESSAGING_SENDER_ID` | Project sender identifier for Cloud Messaging. | `812685909738` | Yes |
| `VITE_FIREBASE_APP_ID` | App registration ID for Firebase web analytics and telemetry. | `1:812685909738:web:ce2e39426aaa4e49bb8193` | Yes |

> [!NOTE]  
> All client-side environment variables in Vite must be prefixed with `VITE_` to be exposed to the client-side code.
